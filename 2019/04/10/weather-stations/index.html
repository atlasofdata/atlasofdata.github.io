A voronoi map of the world based on weather stations.

Colors represents the density of weather stations in an area

We use a Lambert Azimuthal Equal-Area projection so it represents accurately the areas. Weather station data comes from World Clim (2015 data). The 10k+ points dataset was clusterized using supercluster. We use the d3 Geo Voronoito create the vornoi geo clusters of areas based on the dataset.
